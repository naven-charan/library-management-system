# Library Management System

## Overview

A full-stack Library Management System developed to manage and monitor book availability, track user interactions, and streamline library operations.

## Features

* Real-time tracking of book availability
* Book location identification within the library
* Wishlist functionality for users
* Return date tracking and management

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Python

## System Design

The application follows a modular structure separating frontend and backend components. The frontend handles user interaction and display logic, while the backend manages data processing and core functionality.

## Future Enhancements

* Integration with database systems (MySQL / MongoDB)
* User authentication and role-based access
* Deployment as a web application
* REST API development for scalability
